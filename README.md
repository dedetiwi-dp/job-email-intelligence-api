# 📧 Job Email Intelligence API

An AI-powered FastAPI service that extracts structured information from recruiter emails using a local LLM (Ollama).

## ✨ Features

- Extract company name
- Extract job position
- Extract location
- Extract salary
- Detect application status
- Generate a concise summary
- Return structured JSON

## 🛠 Tech Stack

- Python
- FastAPI
- Ollama
- Pydantic
- REST API
